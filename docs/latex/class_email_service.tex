\doxysubsection{Detailed Description}
Сервис для отправки email через SMTP и верификации кодов. 

Использует библиотеку CURL для взаимодействия с SMTP-\/сервером. Коды верификации хранятся в оперативной памяти с ограничением времени жизни.
