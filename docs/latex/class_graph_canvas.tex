\doxysubsection{Detailed Description}
Виджет для отрисовки графика математической функции. 

Отвечает за визуализацию трёх ветвей функции:\+
\begin{DoxyItemize}
\item f₁(x) = √(-\/x) + a (x \texorpdfstring{$<$}{<} 0)
\item f₂(x) = arcsin(x/\+π) + b (0 ≤ x \texorpdfstring{$<$}{<} π)
\item f₃(x) = arccos(x/\+π) -\/ c (x ≥ π) 
\end{DoxyItemize}
