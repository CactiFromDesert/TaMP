Detailed Description

Виджет для отрисовки графика математической функции.

Отвечает за визуализацию трёх ветвей функции:

  • f₁(x) = √(-x) + a (x < 0)
  • f₂(x) = arcsin(x/π) + b (0 ≤ x < π)
  • f₃(x) = arccos(x/π) - c (x ≥ π)
