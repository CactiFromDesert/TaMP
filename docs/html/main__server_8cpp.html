Генерирует JSON-массив точек для построения графика функции.

Функция вычисляет значения f(x) на интервале [-2.0; π+0.5] и возвращает их в формате JSON для отправки клиенту.

Parameters
aКоэффициент a для ветви x < 0 (√(-x) + a)
bКоэффициент b для ветви 0 ≤ x < π (arcsin(x/π) + b)
cКоэффициент c для ветви x ≥ π (arccos(x/π) - c)
numPointsКоличество точек для генерации (по умолчанию 20)
Returns
std::string JSON-строка вида [[x1,y1],[x2,y2],...]

< Нечисловые значения заменяем на null

◆ loop()

void * loop ( void * m)

Потоковая функция для обработки входящих соединений.

Работает в отдельном потоке для каждого клиента. Читает сообщения от клиента и отправляет ответы.

Форматы запросов:

  • "x" → возвращает f(x) с параметрами по умолчанию (a=1, b=2, c=3)
  • "a;b;c" → возвращает JSON со всеми точками графика
Parameters
mУказатель на данные потока (не используется)
Returns
void* Всегда возвращает NULL

Режим 1: вычисление для одного x

Режим 2: генерация графика с параметрами a,b,c
